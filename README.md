# WeatherWise

Wetter-App mit Fokus auf „Was soll ich heute tun/anziehen?“ (Empfehlung + Impact Score + Badges) + optionalem Tagesverlauf und 7‑Tage Forecast.

## Features
- Aktuelles Wetter (OpenWeather)
- Empfehlung + Impact Score (0–100) + Badges
- Tagesverlauf bis Tagesende + 7‑Tage Forecast (Open‑Meteo)

## Setup
1) Repo klonen / Dateien herunterladen
2) `config.example.js` kopieren zu `config.js`
3) In `config.js` deinen OpenWeather API Key eintragen
4) Projekt öffnen (z. B. VS Code Live Server)

> Hinweis: `config.js` ist in `.gitignore` und wird nicht mitgepusht.
